# Intelligent-Invoice-Processing-RPA-UiPath-Google-Sheets-
UiPath-based RPA workflow that automates invoice extraction, validation, and reporting into Google Sheets using AI and Document Understanding.

# Overview
This project is an automated RPA workflow built using UiPath that eliminates manual invoice processing by combining Document Understanding (DU), AI extraction, and workflow automation.

It extracts structured financial data directly from invoices stored in Google Drive and logs it into Google Sheets for real-time business visibility.

# Problem Solved

Manual invoice processing is:
Time-consuming
Error-prone
Difficult to scale

This system automates the entire pipeline from document intake to structured reporting.

# Outcome / Impact
Eliminates manual invoice processing
Improves data accuracy through confidence validation
Enables real-time financial visibility for managers
Helps prioritize payments based on due dates
Scales easily with growing invoice volume
